Progressive Web Apps

precisam ser:

  • Progressivos
  • Responsivos
  • Offilines
  • App-"like"
  • Atualizados

e ainda:

  • Seguros
  • Buscaveis (google.com)
  • Engajáveis
  • Instaláveis
  • Linkaveis

Então vamos ver o que isso tudo

significa:

Mas antes, um lembrete

PWA não é um framework,

é um conceito

Progressivo

Funciona pra todos, se adapta

Responsivo

Múltiplas resoluções

Offilines

Suporta boas ou péssimas redes

App-"like"

Tem uma aparência nativa

Atualizados

Sem app store, sempre atualizados

Seguros

HTTPS, sempre!

Buscáveis

Otimizados para o SEO, search engines... etc.

Engajáveis

Usam-se de features nativas para conquistar o usuário

Instaláveis

Deve ser possível coloca-los na tela inicial do dispositivo

Linkáveis

Cada tela tem um link único facilmente compartilhável.
